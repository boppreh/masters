\textoResumo {Nós apresentamos um algoritmo para o cálculo de quadrados eficiente em $GF(2^m)$ usando bits paralelos. Este algoritmo atinge eficiëncia competitiva, e é adequado para qualquer corpo finito que utilize um polinômio binário irredutível de baixo peso. Para uma grande classe de polinômios irredutíveis este algoritmo proposto é mais eficiente que os melhores algoritmos anteriores. Em contraste, outros "quadradores" eficientes costumam necessitar de uma troca de base ou são adequados apenas para uma pequena porção dos polinômios irredutíveis. Adicionalmente, nós apresentamos um algoritmo simples para redução modular com custo equivalente ao estado da arte. Este algoritmo de redução rápida é usado em nosso quadrador.} 

\palavrasChave {Corpos finitos, teoria de números, polinômios, squaring, criptografia.}

\textAbstract {We present an efficient bit-parallel algorithm for squaring in $GF(2^m)$ using polynomial basis. This algorithm achieves competitive efficiency while being aimed at any choice of low-weight binary irreducible polynomial. For a large class of irreducible polynomials our method is more efficient than the previously best general squarer. In contrast, other efficient squarers often require a change of basis or are suitable for only a small number of irreducible polynomials. Additionally, we present a simple algorithm for modular reduction with equivalent cost to the state of the art for general irreducible polynomials. This fast reduction is used in our squaring method.}

\palavrasChave {Finite field, number theory, polynomial, squaring, cryptography.}
